미디어 관련 요소

이미지 5개 이상 - jpg(jfif), png, gif, webp, svg

동영상 3개 이상 - mp4, webm, ogv

  • video 요소의 태그 속성

    src : 해당 비디오 파일의 경로(위치와 파일명)

    poster : 대기화면(썸네일) 이미지의 파일 경로

    controls : 비디오 재생 제어 패널 표시

    autoplay : 자동재생

    muted : 무음처리(자동 재생시 반드시 필요함)

    loop : 반복재생

    width : 폭, height : 높이

  • 브라우저마다 재생가능한 파일

    mp4, webm, ogv 파일이 브라우저마다 각기 다르게 재생 기능 제공됨

    브라우저MP4WebmOgv
    Internet ExplorerYESNONO
    ChromeYESYESYES
    FireFoxYESYESYES
    SafariYESNONO
    OperaYESYESYES

사운드 3개 이상 - mp3, ogg, wav

  • audio 요소의 태그 속성

    src : 해당 오디오 파일의 경로(위치와 파일명)

    controls : 오디오 재생 제어 패널 표시

    autoplay : 자동재생

    muted : 무음처리(자동 재생시 반드시 필요함)

    loop : 반복재생

  • 브라우저마다 재생가능한 파일

    mp3, wav, ogg 파일이 브라우저마다 각기 다르게 재생기능 제공됨

    브라우저MP3WAVOGG
    Internet ExplorerYESNONO
    ChromeYESYESYES
    FireFoxYESYESYES
    SafariYESYESNO
    OperaYESYESYES
